# StayInCool
A SwiftUI App, which can query area and get weather forecast

## Framework dependency

- WeatherKit
  - config identifier & profile
  - add WeatherKit Capability and Access Wifi Information
- CoreLocation
  -  add an entry to the Info.plist 
- MapKit
